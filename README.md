# Szoo Project
QA testing project for the Szoo application.

## Description
This is a QA testing project created for portfolio purposes.  
The project includes a test plan, manual test cases, and automated tests.

## What I did
- Created a test plan  
- Created manual test cases  
- Found and reported bugs  
- Created automated tests using Cypress  

## Functional Requirements
- Registration, login, and logout (automated tests implemented)  
- Catalog browsing and product selection  
- Product filtering (automated tests implemented)  
- Shopping cart functionality  
- Checkout process (manual testing planned)  
- Navigation through the main menu  
- Responsive behavior on different screen sizes  

## Non-functional Requirements
- Compatibility  
- Design and usability  

## Test Environment
- Environment: Production (or staging, if available)  
- OS: Windows 11  
- Browser: Chrome 136.0.7103.114 (64-bit)  

## Test Artifacts
- Test Plan (Word document)  
- Test Cases (Word document with screenshots)  
- Bug reports (Word document with screenshots)  
- Automated tests (Cypress)
